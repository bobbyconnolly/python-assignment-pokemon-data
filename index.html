Note on setdefault(): The setdefault() method is a handy way to work with dictionaries. It does two things:
  1. If the key doesn't exist in the dictionary, it adds the key with a specified default value.
  2. It returns the value for the key, whether it already existed or was just added.

Example:

my_dict = {'a': 1, 'b': 2}
print(my_dict.setdefault('b', 0))  # Prints: 2 (existing value)
print(my_dict.setdefault('c', 0))  # Prints: 0 (new default value)
print(my_dict)  # Prints: {'a': 1, 'b': 2, 'c': 0}

This method is particularly useful when you want to initialize a list as a value if a key doesn't exist, and then append to that list. For example:

# Declare a new, empty dictionary
pokemon_types = {}

# Use setdefault() to add a new 'Fire' key with an empty list as its value
# 1. If 'Fire' doesn't exist, it's added with [] as the value
# 2. setdefault() returns the value for 'Fire' (new or existing)
fire_pokemon = pokemon_types.setdefault('Fire', [])  

# Append 'Charmander' to the list of Fire-type Pokemon
# fire_pokemon is a reference to the list in the dictionary,
# so this change is reflected in pokemon_types
fire_pokemon.append('Charmander')  

print(pokemon_types)  # Output: {'Fire': ['Charmander']}

# Add another Fire-type Pokemon to show how setdefault() works with existing keys
fire_pokemon = pokemon_types.setdefault('Fire', [])  # 'Fire' already exists
fire_pokemon.append('Charizard')

print(pokemon_types)  # Output: {'Fire': ['Charmander', 'Charizard']}

# Add a Water-type Pokemon to demonstrate setdefault() with a new key
water_pokemon = pokemon_types.setdefault('Water', [])
water_pokemon.append('Squirtle')

print(pokemon_types)  # Output: {'Fire': ['Charmander', 'Charizard'], 'Water': ['Squirtle']}

    4. Note on defaultdict:

    defaultdict is a subclass of Python's built-in dict class. It's used to provide a default value for a nonexistent key, instead of raising a KeyError.

    When you access a key that doesn't exist in a defaultdict, it automatically creates that key with a default value (specified by the function passed to defaultdict). This is particularly useful when you're counting or accumulating values. 

    from collections import defaultdict

# Create a defaultdict with int as the default_factory
d = defaultdict(int)

# Increment counts, even for keys that don't exist yet
words = ['apple', 'banana', 'apple', 'cherry', 'banana', 'date']
for word in words:
    d[word] += 1

print(dict(d))  # Output: {'apple': 2, 'banana': 2, 'cherry': 1, 'date': 1}

    In this example, we don't need to check if a key exists before incrementing its count. The defaultdict automatically initializes the count to 0 for new keys.
